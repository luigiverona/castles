# Code of conduct

Be respectful, constructive, and considerate. Harassment, discrimination, threats, doxxing, and
sharing another person's private data are not tolerated. Maintainers may edit or remove harmful
content and restrict participation to protect the community.

Report conduct concerns privately through the repository maintainer contact path. Do not include
mailbox data, credentials, or security vulnerability details in a conduct report. Enforcement aims
to be proportionate, private, and fair.
